# Quick guide for content editors

1. Sign in to Pages CMS.
2. Open the website repository.
3. Select the `content-review` branch.
4. Choose the page or review dimension.
5. Edit text, lists, images, or links.
6. Set the editorial status.
7. Save the change.
8. Tell the project owner that the content is ready for review.

Please do not edit HTML, CSS, JavaScript, workflows, or `.pages.yml`.
