- Supporting pages
- Forms and giscus settings

## After uploading this revision

1. Replace `.pages.yml` in the repository root with the new one.
2. Upload the rest of the package only if your repository does not already contain the corresponding content files.
3. Commit to the branch that Pages CMS is currently opening.
4. In Pages CMS, close and reopen the repository or refresh the browser.
5. Expand **Mechanisms and all review pages**.
6. Select the mechanism, then open **Review-dimension pages** or **Pillar detail pages**.

Each dimension page edits the full clickable page: definition, explanatory paragraphs, choices, example, stakeholder implications, reviewer questions, evidence requirements, and feedback invitation.
