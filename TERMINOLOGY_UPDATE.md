# Terminology update

This package uses the following public-facing terminology throughout the website and Pages CMS:

- **Electricity System Services Market** replaces **Theoretical Market Framework**.
- **ESSM** replaces the visible acronym **TMF**.
- **Tariff design** replaces **Tariff design mechanism**.

For link stability, internal filenames and URLs remain unchanged, including:

- `content/mechanisms/tmf.yml`
- `pages/theoretical-market-framework.html`
- internal `tmf` keys and code identifiers

This means existing GitHub Pages links and CMS mappings continue to work.
